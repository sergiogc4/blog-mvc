# Blog MVC - Projecte PHP OOP

## Instal·lació
1. Clonar el repositori
2. `composer install`
3. Configurar `config/database.php`
4. Importar `database/migrations/01_create_tables.sql`
5. `php -S localhost:8000 -t public`
